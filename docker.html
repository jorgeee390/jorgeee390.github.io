Docker

Docker es un subsistema de virualización que comparte con el anfitración el kernel y el sistema de archivos. El anfitrión siempre es un sistema linux. Si estamos en Windows o MAC, necesitaré una maquina virtual Windows virtualizada con virualbox o similar

Un software de maquinas vituales necesita instalar el sistema operativo completo. Ejemplos de sistemas de virtualización:

  • Virtualbox (Para múltiples plataformas)
  • Hyper-V (Para Windows)
  • Parallels (Para MAC)
  • HMware (Para múltiples plataformas)
  • qEMV (Para Linux)
  • XEN (Para UNIX/Linux)
  • KUM (Para UNIX/Linux)

Un software de contenedores es un sistema de virtualización que comparte el kernel con el alfitrión. Se basa en el concepto de microservicio, donde tengo todos los programas instalados, configurados y funcionando, gracias a los contenedores. Ejemplo de software de contenedores:

  • LXD
  • Docker
  • Kubernetes

Dockerfile

Un Dockerfilees una receta para crear una imagen para una aplicación particular de Docker. de este modo podemos tomar una imagen base, instalarte programas, configurar esos progamas y dejarla preparada para crear contenedores a partir de ella.
